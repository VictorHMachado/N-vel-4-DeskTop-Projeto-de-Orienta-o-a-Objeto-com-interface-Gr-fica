Projeto do Desktop Nível 4 para a matéria de Orientação a Objeto com o Professor Henrique.

Nesse repositório existem dois arquivos, o arquvio dados_cartas4.py que é o sistema necessário para criar o banco de dados para o projeto usando sklite. Além desse arquivo, existe também o arquivo principal cartas4.py que é o arquivo para rodar de fato o simulador de cartas, esse projeto faz uso do CustomTkinter, uma extensão que dispolibiliza a criação de Interface Gráfica em python
